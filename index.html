The Wuxi Finger Hold

fs gh helps easily navigate your repository in Github. 

$ fs gh
# opens current repo on github

$ fs gh pulls (fs gh p)
# opens current repo's pull requests on github

$ fs gh issues (fs gh i)
# opens current repo's issues on github

$ fs gh pulls new (fs gh p n)
# opens new pull request in current repo on github

$ fs gh issues new (fs gh i n)
# opens new issue in current repo on github

$ fs gh pulls closed (fs gh p c)
# opens current repo's closed pull requests on github

$ fs gh issues closed (fs gh i c)
# opens current repo's closed issues on github

$ fs gh search (fs gh s) <query>
# search in current repo

The Pool of Sacred Tears

fs setup bootstraps any project you want. Any. 

$ fs setup rails <project name>
# bootstraps new rails project based on rails-base template

$ fs setup api <project name>
# bootstraps new rails project based on rails-base-api template

$ fs setup static <project name>
# bootstraps static site on middleman based on static-base template
